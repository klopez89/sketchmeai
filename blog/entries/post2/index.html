Organize Your Generations with Collections

Tuesday, April 23rd 2024

 

As of April 12th, we released the new Collections feature that enables you to better organize your SketchMeAi generations. Instead of just having the default collection, you can now create, rename and swap between collections.

 

Accessing these features varies between desktop and mobile web experiences. On desktop, you will see the 'Collections' button above the generations grid. The button will also include the name of the currently selected collection. Clicking on the button will reveal a dropdown menu where you can select from: New Collection, Change Collection, and Rename Collection.

The modals are consistent across desktop + mobile web but the access point is different on mobile web.

 

On mobile web, you can access these collection-related actions from the floating generation menu. Specifically, the button w/ the up arrow, which we call the Generation Actions button. Tapping on that button will reveal a dropdown similar to that on desktop.

This dropdown will be where you can see the current collection name, and a new action that was also added in this update: Multi-Select.

 

Tapping 'Multi-Select' will send your generation grid into select mode, where each generation is tappable for select/deselect. A sticky select menu will also appear at the bottom of your screen and it enables you to take two different actions: Delete and Share.

Pressing 'Delete' will permanently remove the selected generations from our database so be careful when taking this action. Pressing 'Share' will bring up your browser's native share window so that you can select where you want to share your selected generations.

The last feature included in this update is Favoriting! Now you can tap the heart icon on the bottom left corner of a generation to mark it as a favorite. This can be helpful in finding your best generations and removing the ones you don't like w/ the multi-select + delete feature.

We hope that with these features, you're inspired to generate and manage images across different themes and then share your favorites with your friends!
